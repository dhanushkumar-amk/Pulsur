# Security Policy

## Supported Versions

The following versions of Pulsur are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of Pulsur seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not open a GitHub issue.** This would make the vulnerability public before we have a chance to fix it.
2. Send an email to **security@pulsur.dev** (Note: this is a placeholder address).
3. Include as much information as possible, including:
   - A descriptive title.
   - Steps to reproduce the issue.
   - A proof-of-concept (POC) if possible.
   - Potential impact of the vulnerability.

We will acknowledge your report within 48 hours and provide a timeline for the fix. Once the fix is released, we will give you credit for the discovery (if you wish).
